Plantilla base para GitHub Pages

Diseños limpios, código ordenado y listo para personalizar.

Esta estructura combina buenas prácticas en HTML, CSS y JavaScript para que puedas iterar rápido sobre una base elegante. Ideal para portafolios, landings o documentación visual.
