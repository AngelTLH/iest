Plantilla base para GitHub Pages

Diseños limpios, código ordenado y listo para personalizar.

Esta estructura combina buenas prácticas en HTML, CSS y JavaScript para que puedas iterar rápido sobre una base elegante. Ideal para portafolios, landings o documentación visual.

Explorar secciones Hablar ahora

Estructura sugerida

  • assets/css para estilos globales.
  • assets/js para comportamientos ligeros.
  • assets/img para gráficas o íconos.
  • Componentes semánticos listos para adaptar.

Orden primero, creatividad después.

Pilares de la plantilla

Bloques reutilizables con visual sólido.

Cada bloque está pensado para ser fácil de duplicar o eliminar. Los estilos usan variables y clases atómicas para que puedas combinarlos sin romper el diseño.

Maquetación semántica

Secciones claras con etiquetas HTML adecuadas y jerarquía tipográfica que facilita la lectura en escritorio y móvil.
